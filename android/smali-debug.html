# Android Smali 调试

# 1. 使用 apktool 对 apk 文件解包

apktool -d cloud189_v8.5.8.apk
1

# 2. 安装 ideasmali 插件

Android Studio 要想调试 smali 需要安装这个插件

下载地址：https://bitbucket.org/JesusFreke/smali/downloads/?tab=tagsopen in new window

# 3. 导入项目

打开 Android Studio，选择 Import project，选择第一步解包出的代码目录导入。

# 4. 启动 app

使用 USB 连接手机设备，打开欲调试 app 界面，使用下面的指令获取对应包名

adb shell dumpsys activity top | grep --color=always ACTIVITY
1

获取出包名后就可以以调试模式启动应用了

adb shell am start -D -n com.cn21.ecloud/cn.com.chinatelecom.account.lib.base.ui.AuthActivity
1

# 5. 附加进程

启动进程后，在 Android Studio 菜单栏找到 Attach Debugger to Android Process 图标，附加源码到进程中即可开始调试。
