# Grid 网格布局

Grid 网格布局方案与最早的表格布局方案极其相似，它擅长把一个页面分成若干个格子，以及控制这些格子的大小、位置、层次等关系。有很多复杂的界面可以很轻松的通过它来实现。

# 一、基本概念

# 1.1 网格容器

通过使用 display: grid 或 display: inline-grid 可以创建一个网格容器。

在这个例子中，我使用一个类名为 wrapper 的 div 元素作为网格容器，在它内部的三个元素是网格的项目。

<div class="wrapper">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

# 1.4 repeat() 标记

使用 repeat() 标记可以让我们更方便的重复多个相同的值。如：

.wrapper {
  display: grid;
  grid-template-columns: 100px 100px 100px 100px;
}

.wrapper {
  display: grid;
  grid-template-columns: repeat(4, 100px);

  /*
    重复多个：grid-template-columns: repeat(2, 100px 100px);
    作为一部分：grid-template-columns: 100px repeat(2, 100px) 100px;

}

在设置网格时，有时候需要限定格子的最小或最大尺寸。此时可以使用 minmax() 标记。它和 CSS 中的 min、max 指令使用类似。

# 二、容器属性

# 2.1 grid-template-rows

在设置网格容器后，需要指定容器每行的高度。这时可以使用 grid-template-rows 指令来指定。相关示例代码请参照第 2.2 章节。

# 2.5 grid-template-areas

使用 grid-template-areas 指令在网格中划分区域，单个区域由一个或多个格子构成。使用区域的方式可以很轻松的实现某些布局。

<section id="page">
  <header>Header</header>
  <nav>Navigation</nav>
  <main>Main area</main>
  <footer>Footer</footer>
</section>
1
2
3
4
5
6
#page {
  display: grid;
  width: 100%;
  height: 250px;
  grid-template-areas:
    'head head'
    'nav  main'
    'nav  foot'; /* 1.区域划分 当前为 三行 两列 */
  grid-template-rows: 50px 1fr 30px; /* 2.各区域 宽高设置 */
  grid-template-columns: 150px 1fr;
}
#page > header {
  grid-area: head; /* 3. 指定当前元素所在的区域位置, 从grid-template-areas选取值 */
  background-color: #8ca0ff;
}
#page > nav {
  grid-area: nav;
  background-color: #ffa08c;
}
#page > main {
  grid-area: main;
  background-color: #ffff64;
}
#page > footer {
  grid-area: foot;
  background-color: #8cffa0;
}

# 三、项目属性

# 3.1 grid-row

指定项目在网格中行的位置，使用 grid-row 指令。它是 grid-row-start 与 grid-row-end 的简写形式。相关示例代码请参照第 3.2 章节。

# 3.2 grid-column

指定项目在网格中行的位置，使用 grid-column 指令。它是 grid-column-start 与 grid-column-end 的简写形式。

<div class="wrapper">
  <div>1</div>
  <div class="two">2</div>
  <div class="three">3</div>
  <div class="four">4</div>
</div>
1
2
3
4
5
6
.wrapper {
  width: 300px;
  height: 100px;
  border: 1px solid #dadada;

  display: grid;
  grid-template: repeat(5, 1fr) / repeat(5, 1fr);
}
.wrapper div {
  margin: 1px;
  color: white;
  background: #ff9191;
}

.wrapper .two {
  background: #2196f3;
  grid-row: 1 / 3;
  /*等同：grid-row: 1 / span 2;*/
  /*等同：grid-row:     span 2;（省略为当前位置）*/

  grid-column: 2 / 3;
  /*等同：grid-column: 2 / span 1;*/
}

.wrapper .three {
  background: #9e9e9e;

  grid-row-start: 3;
  grid-row-end: span 1;
  /*等同：grid-row: 3 / span 1;*/

  grid-column-start: 3;
  grid-column-end: span 1;
  /*等同：grid-column: 3 / span 1;*/
}

.wrapper .four {
  background: #ff6a00;
  grid-area: 5 / 1 / span 1 / span 5;

  /*等同：
    grid-row: 5 / span 1;
    grid-column: 1 / span 5;
  */
}
