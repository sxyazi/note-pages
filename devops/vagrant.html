        # Vagrant

        使用 Vagrant 可以在无需操作系统镜像文件的情况下，为不同虚拟机软件快速安装不同的系统环境。

        # 安装

        brew install --cask vagrant
brew install --cask virtualbox  # vagrant依赖

        # 创建操作系统

        可以透过 Vagrant Cloud (opens new window) 页面查找你需要的操作系统。

        # 初始化 Vagrantfile 文件
vagrant init ubuntu/groovy64
# 启动虚拟机
vagrant up

        # 常用命令

        注意大部分的命令都需要在 Vagrantfile 文件的所在目录下执行。

        • 连接 SSH vagrant ssh
        • 停止虚拟机 vagrant halt
        • 恢复虚拟机 vagrant resume
        • 销毁虚拟机 vagrant destroy
