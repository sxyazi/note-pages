          # Objective-C Block

          在 OC 中，Block 是一种数据类型。它与函数类似，可以使一段代码被反复调用，与其它语言中的「匿名函数」很像。

          # 基础使用

          int (^sum) (int, int);
sum = ^(int a, int b) {
    return a + b;
};

NSLog(@"%d", sum(3, 22));

          也可以连起来写：

          int (^sum) (int, int) = ^(int a, int b) {
    return a + b;
};

NSLog(@"%d", sum(3, 22));

          无参数可以省略 ^ 后面的括号：


           





          int (^sum)(void);
sum = ^{
    return 25;
};

NSLog(@"%d", sum());

          # 应用场景

          #import <Foundation/Foundation.h>

void doSomething(void (^block)(void))
{
    NSLog(@"准备……");
    block();
    NSLog(@"完成……\n");
}

int main(int argc, const char * argv[]) {
    doSomething(^{
        NSLog(@"吃饭");
    });

    doSomething(^{
        NSLog(@"睡觉");
    });

    doSomething(^{
        NSLog(@"打豆豆");
    });

    return 0;
}

          # 注意事项

          • 在 block 中可以访问外界变量；

          • 在 block 中，可以定义与外界同名的变量，优先访问内部的变量；

          • 默认情况下，不能在 block 中修改外界变量。如果想要修改，需要在变量前面加上 __block。不加 __block 变量是值传递到 block 中，加上是指针传递到 block 中。

            __block int a = 10;

          • 默认情况下，block 存储在栈中，此时不会对所访问的外界对象进行 retain 操作；如果对 block 进行 copy，其会转移到堆中，此时会对所访问的外界对象 retain 一次，但若该对象含有 __block 标记则不会 retain。

            void (^test)(void) = ^{ p; };
Block_copy(test);

Person *p = [Person new];
NSLog(@"retainCount: %lu", [p retainCount]);  // 1
test();
NSLog(@"retainCount: %lu", [p retainCount]);  // 2

          • 为了保证对象的正常释放，如果需要在 block 中访问外界对象时，一定要给该对象加上 __block 标记。

            __block void (^test)(void) = ^{ p; };
Block_copy(test);

Person *p = [Person new];
NSLog(@"retainCount: %lu", [p retainCount]);  // 1
test();
NSLog(@"retainCount: %lu", [p retainCount]);  // 1

          # 与 typedef 结合

          定义：

          typedef int (^sumBlock) (int, int);

          使用：

          sumBlock sum = ^(int a, int b) {
    return a + b;
};

NSLog(@"%d", sum(3, 22));
