# 集合

集合的特征：确定性、互异性、无序性

若 aaa 是 AAA 集合中的元素，记为 a∈Aa\in{A}a∈A

若 aaa 不是 AAA 集合中的元素，记为 a∉Aa\notin{A}a∈/A

两个集合 AAA、BBB，若 ∀x∈A\forall{x}\in{A}∀x∈A 都有 x∈Bx\in{B}x∈B，则称 AAA 是 BBB 的子集，记为 A⊂BA\subset{B}A⊂B
