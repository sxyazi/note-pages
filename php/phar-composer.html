              # 打包 Composer 依赖为 Phar 文件

              # 创建空项目

              mkdir my-project

              # 安装欲打包依赖

              cd my-project
composer require guzzlehttp/guzzle

              # 添加 index.php 文件

              在项目根目录添加 index.php 文件用以加载 Composer 的 autoload 代码。

              <?php require __DIR__ . '/vendor/autoload.php';

              # 压缩整个项目（可省略）

              由于直接打包出来的 phar 文件较大，所以需要对下载下来的 Composer 依赖库代码进行压缩，并去除注释、说明、版权等文件。

              <?php

const REMOVED_FILES = [
  'license',
  'makefile',
  'dockerfile'
];

$rii = new RecursiveIteratorIterator(new RecursiveDirectoryIterator($argv[1]));

foreach ($rii as $file) {
  if ($file->isDir()) {
    continue;
  }

  $path = $file->getPathname();
  list($name, $ext) = [basename($path), strtolower(pathinfo($path, PATHINFO_EXTENSION))];

  if (in_array($ext, ['md'])) {
    echo "Removeing {$path}\n";
    unlink($path);
    continue;
  }

  if (in_array($name, ['.DS_Store'])) {
    echo "Removeing {$path}\n";
    unlink($path);
    continue;
  }

  if (in_array(strtolower($name), REMOVED_FILES)) {
    echo "Removeing {$path}\n";
    unlink($path);
    continue;
  }

  if (in_array($ext, ['php'])) {
    echo "Minifying {$path}\n";
    file_put_contents($path, php_strip_whitespace($path));
  }

  if (in_array($ext, ['json']) || in_array(strtolower($name), ['composer.lock'])) {
    echo "Minifying {$path}\n";
    file_put_contents($path, json_encode(json_decode(file_get_contents($path))));
  }
}

              保存上面代码为 minify.php 然后执行下方代码压缩整个项目：

              php minify.php my-project

              # 打包依赖

              第一次使用需要先安装 phar-composer (opens new window) 工具：

              curl -JOL https://clue.engineering/phar-composer-latest.phar

              开始打包：

              php phar-composer.phar build my-project

              打包完成后会在当前目录生成一个与目录名相同的 .phar 文件，这里是 my-project.phar。

              # 使用 Phar

              最终打包出的 Phar 文件在 PHP 中直接 require 即可使用。

              <?php
require 'my-project.phar';
$client = new GuzzleHttp\Client();

var_dump($client);
