        # frp

        之前写过 Ngrok 内网穿透工具，但是经过一段时间使用下来发现问题比较多。经常需要重启服务才能连接成功，所以我就转移到了 frp 这款软件。

        GitHub 地址：https://github.com/fatedier/frp (opens new window)

        # 服务端

        服务端用于提供中转服务，下载对应系统的发行版本

        wget https://github.com/fatedier/frp/releases/download/v0.35.1/frp_0.35.1_linux_amd64.tar.gz
tar vfxz frp_0.35.1_linux_amd64.tar.gz && cd frp_0.35.1_linux_amd64

        编辑 frps.ini 配置文件

        [common]
bind_port = 7000
vhost_http_port = 7001
vhost_https_port = 7002
token = <secret>

        启动服务

        ./frps -c frps.ini > /dev/null 2>&1 &

        # 客户端

        由于我是 macOS 系统，所以选择的文件是 frp_0.35.1_darwin_amd64.tar.gz (opens new window)

        解压后修改 frpc.ini 配置文件

        [common]
server_addr = 1.2.3.4
server_port = 7000
token = <secret>

[git]
type = tcp
local_ip = 192.168.1.33
local_port = 22
remote_port = 53395

[gogs]
type = http
local_ip = 192.168.1.33
local_port = 3000
custom_domains = gogs.yz

        启动客户端

        ./frpc -c frpc.ini

        # 完成

        最终将绑定的域名解析到中转服务器 IP 即可，或者采用更加方便的修改 hosts 文件方式

        1.2.3.4 gogs.yz

        对应的映射结果

        • git: tcp://gogs.yz:53395
        • gogs: http://gogs.yz:7001
