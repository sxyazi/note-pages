# IDEA

# 主题

安装 Material Theme UIopen in new window，我目前在使用的是其中的 Atom One Dark Theme。

再安装 Atom Material Iconsopen in new window，这是一套图标，让文件列表显示得更漂亮。

# 编辑区字体

配置路径：Preferences | Editor | Font

Font 设置 Menlo，Size 16，Line Spacing 为 1.6

# 终端字体

配置路径：Preferences | Editor | Color Scheme | Console Font

Font 设置 Menlo，Size 14，Line Spacing 为 1.8

# 自动保存

配置路径：Preferences | Appearance & Behavior | System Settings

在 Autosave 区域取消默认勾选 Save files when switching to a different application

# 代码风格

配置路径：Preferences | Editor | Code Style

# 常用插件

# Laravelopen in new window

安装 Laravel 插件后在 Preferences | Languages & Frameworks | PHP | Laravel 勾选 Enable plugin for this project 和 Use AutoPopup for completion

# JS GraphQLopen in new window

提供对 GraphQL 语言的高亮、自动补全支持

# Rainbow Bracketsopen in new window

彩虹括号。使用不同的颜色表示每一对括号，便于区分。

# EnvFile、.​env files support

  • EnvFileopen in new window
  • .​env files supportopen in new window

提供对 .env 文件的高亮、自动补全支持

# Save Actionsopen in new window

在 Preferences | Save Actions 中勾选 Activate save actions on save、Optimize imports、Reformat file。

# IdeaVim、IdeaVimExtension

  • IdeaVimopen in new window
  • IdeaVimExtensionopen in new window

IdeaVim 是 IntelliJ 平台 IDE 的 Vim 仿真插件。配合 IdeaVimExtension 插件可以实现输入法自动切换。

# IdeaVimExtension 配置
set keep-english-in-normal
set nokeep-english-in-normal-and-restore-in-insert

# Command Line Toolopen in new window

安装后在 Preferences | Tools | Command Line Tool Support，点击 Add，Choose Tool 选择 Laravel，Visibility 选择 global

选择对应版本的 PHP 解释器和 artisan 文件路径，然后点击 OK 保存设置。

# WakaTimeopen in new window

记录你的编码时间。在其网站上可以友好的使用柱状图、饼状图统计你的编码时长。并且会每周向你的电子邮箱发送一封周汇总报告。

# IdeaVim

常用按键，参考自《提升 Android 開發速度系列(5) Vim》open in new window

cw      从游标位置开始变更单字
ciw     变更整个单字
cc      变更整行
c$      从游标位置开始变更该行

s       删除游标后字元并进入插入模式
S       删除整行并进入插入模式

r       变更游标的字
J       将下一行合并至该行
gJ      将下一行合并至该行，但保留下一行的前面的空白

fa      跳至字元 a 下次出现的位置
ta      跳至字元 a 下次出现位置的上一个字元
Fa      往前跳至字元 a 下次出现的位置
Ta      往前跳至字元 a 下次出现位置的上一个字元

dfa     删除到下一个a出现的位置
dta     删除到下一个a出现的前一位置
cfa     删除到下一个a出现的位置，并开始编辑
cta     删除到下一个a出现的前一位置，并开始编辑
.       重复执行上个命​​令

aw      选取该单字
ab      选取 () 及所包围的区块
aB      选取 {} 及所包围的区块
ib      选取 () 内的区块
iB      选取 {} 内的区块
