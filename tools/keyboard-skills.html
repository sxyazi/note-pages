# 键盘技巧

# Karabiner Elements

使用 Karabiner Elements 将不常用的右 Command 键映射为 F19 超级键。

{
  "title": "Take advantage of right command key",
  "rules": [
    {
      "description": "Change right_command key to command+control+option+shift. (Post f19 key when pressed alone)",
      "manipulators": [
        {
          "type": "basic",
          "from": {
            "key_code": "right_command",
            "modifiers": {
              "optional": ["any"]
            }
          },
          "to": [
            {
              "key_code": "left_shift",
              "modifiers": ["left_command", "left_control", "left_option"]
            }
          ],
          "to_if_alone": [
            {
              "key_code": "f19"
            }
          ]
        }
      ]
    }
  ]
}

# Alfred

将 Alfred 的呼出快捷键设置为 F19 超级键。

# Thor

使用 Thor 设置常见软件快速切换按键，以下是我的配置：

  • iTerm2 设置为 Option-1
  • Google Chrome 设置为 Option-2
  • Visual Studio Code 设置为 Option-Q
  • IntelliJ IDEA 设置为 Option-W
