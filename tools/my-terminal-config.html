# 我的终端配置

# iTerm2

iTerm2 是 macOS 系统终端的替代方案，相比系统自带的终端程序更加易于扩展使用。

# 窗口

在 Appearance -> General 面板中的 Theme 选择 Compact 模式。

切换到 Profiles -> Window 面板，根据自己喜好调整 Transparency 到适合位置。

# 字体

使用 Command + , 打开设置，切换到 Profiles -> Text 面板。

Cursor 设置为 Vertical bar，Font 设置为 16pt Monaco，Vertical 设置为 180%

# 配色

配色使用 Dracula，下载 iTerm2 版本的 配色文件open in new window 导入。

导入方式：Profiles -> Colors 面板中点击 Color Presets… 下拉菜单，点击 Import… 选项。

# 快捷键

在 Profiles -> Keys 面板中设置如下。

Option + Left 向前一个单词：

  • Keyboard Shortcut: ⌥←
  • Action: Send Escape Sequence
  • Esc+: b

Option + Right 向后一个单词：

  • Keyboard Shortcut: ⌥→
  • Action: Send Escape Sequence
  • Esc+: f

Option + Delete 删除一个单词：

  • Keyboard Shortcut: ⌥⌫
  • Action: Send Hex Code
  • 0x17

# 其它偏好

重用会话目录

在 Profiles -> General 面板中找到 Working Directory 选项，选中其中的 Reuse previous session's directory 。

# Fish Shell

Fish 是一个智能且友好的 Shell 程序，用来替代系统默认的 bash。

# 安装

brew install fish

echo "/usr/local/bin/fish" | sudo tee -a /etc/shells
chsh -s /usr/local/bin/fish
1
2
3
4

使用上面的指令安装并且设置为默认的交互 shell，重启终端会话。

# 主题

执行 fish_config 可以打开 fish 自带的一个网页端配置程序。在 colors tab 中选择 Dracula 主题点击 Set Theme 按钮保存，然后切换到 prompt tab 选择 Robbyrussell 保存。

# Alias

在 ~/.config/fish/config.fish 文件中加入

alias p="pwd"
alias l="ls -al"
alias cat="ccat"

# Fisher

Fisher 是 fish 的一个包管理器，类似的还有 oh-my-fish。

curl -sL https://git.io/fisher | source && fisher install jorgebucaran/fisher
1

常用插件：

  • 0rax/fish-bdopen in new window
  • PatrickF1/fzf.fishopen in new window
  • jhillyerd/plugin-gitopen in new window
  • oh-my-fish/plugin-osxopen in new window
  • oh-my-fish/plugin-proxyopen in new window

# Autojump

众所周知的辅助 cd 导航工具。首先先用 brew 安装 autojump：

brew install autojump
1

之后在 ~/.config/fish/config.fish 中加入如下代码引用 autojump 自带的 fish 文件：

[ -f /usr/local/share/autojump/autojump.fish ]; and source /opt/homebrew/share/autojump/autojump.fish

# For ARM
[ -f /opt/homebrew/share/autojump/autojump.fish ]; and source /opt/homebrew/share/autojump/autojump.fish
1

3
4

注：autojump.fish 文件的位置与 autojump 的安装方式有关，如使用 brew 安装则可以使用 brew ls --verbose autojump 查询到。
