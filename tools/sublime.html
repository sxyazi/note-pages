# Sublime 常用插件

  • Emmet：助力 HTML/CSS 开发
  • Alignment：自动对其多行
  • FileDiffs：文件差异对比
  • DocBlockr：更好的写块注释
  • CodeFormatter：提供代码格式化
  • ConvertToUTF8：文件转换 UTF8 编码
  • SFTP
  • SublimeCodeIntel：代码智能自动完成
  • SideBarEnhancements：侧边栏扩展
  • Delete Current File：添加“删除当前文件”命令
  • SublimeLinter：代码风格限制插件

# 快捷键

  • Cmd-Ctrl-G：选择所有匹配的项目
  • Cmd-Shift-L：使选中行进入多行编辑模式
